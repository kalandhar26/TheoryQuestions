# Service Discovery
- Mechanism for services to dynamically find and communicate with each other without hardcoding IPs.

## Types
- Client-side discovery (client decides)
- Server-side discovery (load balancer decides)

## Widely Used Tools
Eureka
Consul
Zookeeper
Kubernetes (built-in discovery)

## Banking Reality
Old Spring stack → Eureka
Infra-heavy systems → Consul / Zookeeper
Modern → Kubernetes (DNS-based discovery)

## Blunt Gaps
Service registry failure = system-wide outage
Stale entries → traffic routed to dead instances
Needs heartbeat + health checks (people forget this)
