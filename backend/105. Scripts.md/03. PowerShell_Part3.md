## Variables: Automatic & User-Created

### Automatic Variables (PowerShell-Managed)

- PowerShell provides automatic variables that are created and maintained by the engine :

| Variable                             | Purpose                          |
|--------------------------------------|----------------------------------|
| `$_` / `$PSItem`                     | Current object in pipeline       |
| `$?`                                 | Success status of last operation |
| `$^`                                 | First token of last command      |
| `$$`                                 | Last token of last command       |
| `$args`                              | Undeclared parameters array      |
| `$Error`                             | Array of recent errors           |
| `$HOME`                              | User's home directory            |
| `$PSScriptRoot`                      | Directory of executing script    |
| `$PSCommandPath`                     | Full path of executing script    |
| `$IsWindows`, `$IsLinux`, `$IsMacOS` | OS detection (PowerShell 7+)     |
| `$LASTEXITCODE`                      | Exit code of last native command |
| `$PSVersionTable`                    | PowerShell version info          |

```powershell
# OS-specific commands using automatic variables
if ($IsLinux)
{
    df -h  # Linux disk free
}
elseif ($IsWindows)
{
    Get-PSDrive | Where-Object { $_.Free -gt 0 }
}
```

## Operators

### Comparison Operators

```text
    # Basic comparison (case-insensitive by default)
    -eq    # Equal to
    -ne    # Not equal to
    -gt    # Greater than
    -ge    # Greater than or equal
    -lt    # Less than
    -le    # Less than or equal
    
    # Case-sensitive variants
    -ceq, -cne, -cgt, -cge, -clt, -cle
    
    # Case-insensitive explicit
    -ieq, -ine, -igt, -ige, -ilt, -ile
    
    # Examples
    5 -eq 5           # True
    "Power" -eq "power"   # True (case-insensitive)
    "Power" -ceq "power"  # False (case-sensitive)
```
### Type Operators

```text
# Type comparison
-is      # Is of type
-isnot   # Is not of type
-as      # Convert to type

# Examples
"hello" -is [string]       # True
123 -is [int]              # True
Get-Date -is [DateTime]    # True
"123" -as [int]            # 123 (converted)
```

### Like & NotLike (Wildcard Matching)

```text
-like       # Wildcard match
-notlike    # Wildcard no match

# Wildcards: * = multiple chars, ? = single char, [a-z] = range
"PowerShell" -like "Power*"      # True
"file.txt" -like "file.???"      # True
"server01" -like "server[0-9][0-9]"  # True

# Practical example
Get-Service | Where-Object { $_.Name -like "Win*" }
```

### Logical Operators

```text
-and    # Both conditions true
-or     # At least one true
-xor    # Exclusive OR (one true, not both)
-not # Negation
!# Negation (alternative)

# Examples
($age -ge 18) -and ($age -le 65)    # Working age
($status -eq "Running") -or ($status -eq "Starting")  # Active states
-not $isDisabled                     # Enabled
!$isNull                            # Not null
```

### String Operators & Sub-Expressions

```powershell
# String concatenation
$first = "Power"
$second = "Shell"
$full = $first + $second           # PowerShell
$full = "$first$second"            # PowerShell (expansion)

# Sub-expressions $()
$service = Get-Service -Name w32time
"The service $( $service.Name ) is $( $service.Status )"

# Here-strings (multi-line)
$query = @"
SELECT * FROM Users
WHERE Status = 'Active'
AND CreatedDate > '$((Get-Date).AddDays(-30).ToString("yyyy-MM-dd") )'
"@

# String formatting (-f operator)
"User: {0,-10} | Status: {1,8} | ID: {2:D4}" -f "John", "Active", 42
# Output: User: John       | Status:   Active | ID: 0042

# Format specifiers
"{0:C}" -f 1234.50        # Currency: $1,234.50
"{0:P}" -f 0.85           # Percent: 85.00%
"{0:N2}" -f 1234.5678     # Number: 1,234.57
"{0:D8}" -f 42            # Decimal: 00000042
"{0:X}" -f 255            # Hex: FF
```

## Conditional & Switch Statements

### If/ElseIf/Else

```powershell
$cpu = Get-Counter '\Processor(_Total)\% Processor Time'
$value = $cpu.CounterSamples.CookedValue

if ($value -gt 90)
{
    Write-Host "CRITICAL: CPU at $value%" -ForegroundColor Red
    # Send alert, throttle processes, etc.
}
elseif ($value -gt 75)
{
    Write-Host "WARNING: CPU at $value%" -ForegroundColor Yellow
}
elseif ($value -gt 50)
{
    Write-Host "ELEVATED: CPU at $value%" -ForegroundColor Cyan
}
else
{
    Write-Host "NORMAL: CPU at $value%" -ForegroundColor Green
}
```

### Ternary Operator

```powershell
# Traditional
if ($status -eq "Running")
{
    $color = "Green"
}
else
{
    $color = "Red"
}

# Ternary (PowerShell 7+)
$color = ($status -eq "Running") ? "Green" : "Red"

# Null-coalescing (PowerShell 7+)
$configPath = $env:CONFIG_PATH ?? "C:\Default\Config.json"

# Null-conditional assignment (PowerShell 7.1+)
$logLevel = $settings.LogLevel ?? = "Information"
```

### Switch Statements

```powershell
# Basic switch
$day = 3
switch ($day)
{
    1 {
        "Monday"
    }
    2 {
        "Tuesday"
    }
    3 {
        "Wednesday"
    }
    4 {
        "Thursday"
    }
    5 {
        "Friday"
    }
    { $_ -gt 5 } {
        "Weekend"
    }
    default {
        "Invalid day"
    }
}

# Switch with wildcards and regex
$filename = "report_2024_Q1.pdf"
switch -Wildcard ($filename)
{
    "*.pdf"  {
        "PDF Document"
    }
    "*.doc*" {
        "Word Document"
    }
    "*.xls*" {
        "Excel Spreadsheet"
    }
    "report_*" {
        "Report File"
    }
}

# Switch with regex and capture groups
$logEntry = "Error: 2024-03-15 14:30:22 Connection timeout"
switch -Regex ($logEntry)
{
    "(\d{4}-\d{2}-\d{2})" {
        "Date found: $( $matches[1] )"
    }
    "(\d{2}:\d{2}:\d{2})" {
        "Time found: $( $matches[1] )"
    }
    "(Error|Warning|Info)" {
        "Level: $( $matches[1] )"
    }
}

# Switch on file content (line by line)
switch -File C:\Logs\app.log
{
    "Error*" {
        Write-Host $_ -ForegroundColor Red
    }
    "Warning*" {
        Write-Host $_ -ForegroundColor Yellow
    }
    default {
        Write-Host $_ -ForegroundColor Gray
    }
}
```

## Creating Dynamic Scripts

```powershell
[CmdletBinding()]
param(
    [Parameter(Mandatory = $true, Position = 0)]
    [string]$ComputerName,

    [Parameter()]
    [ValidateSet("Running", "Stopped", "All")]
    [string]$Status = "All",

    [Parameter()]
    [int]$MaxResults = 50,

    [Parameter()]
    [switch]$IncludeSystem
)

# Dynamic script logic based on parameters
$filter = if ($Status -eq "All")
{
    "*"
}
else
{
    $Status
}
Get-Service -ComputerName $ComputerName |
        Where-Object { $_.Status -like $filter } |
        Select-Object -First $MaxResults
```

### Dynamic Code Execution

```powershell
# Invoke-Expression (use with caution)
$command = "Get-Process | Where-Object {`$_.CPU -gt 100}"
Invoke-Expression $command

# Script blocks for dynamic execution
$block = {
    param($name, $threshold)
    Get-Process -Name $name | Where-Object { $_.WorkingSet64 -gt $threshold }
}
& $block -name "chrome" -threshold 500MB

# Dynamic module loading
$moduleName = "ActiveDirectory"
if (-not (Get-Module -ListAvailable -Name $moduleName))
{
    Install-Module $moduleName -Force
}
Import-Module $moduleName
```

## Functions, Input & Scope

```powershell
# Simple function
function Get-MemoryUsage
{
    Get-Process |
            Select-Object Name,
            @{ N = 'MemoryMB'; E = { [math]::Round($_.WorkingSet64 / 1MB, 2) } }
}

# Function with parameters
function Stop-MyProcess
{
    param(
        [string]$Name,
        [switch]$Force
    )

    $proc = Get-Process -Name $Name -ErrorAction SilentlyContinue
    if ($proc)
    {
        if ($Force)
        {
            $proc | Stop-Process -Force
        }
        else
        {
            $proc | Stop-Process
        }
        "Stopped $Name"
    }
    else
    {
        "Process $Name not found"
    }
}
```

### Advanced Functions (Cmdlet-Style)

```powershell
function Get-SystemHealth
{
    [CmdletBinding(
            SupportsShouldProcess = $true,
            ConfirmImpact = 'Medium'
    )]
    [OutputType([PSCustomObject])]
    param(
        [Parameter(
                Mandatory = $true,
                Position = 0,
                ValueFromPipeline = $true,
                ValueFromPipelineByPropertyName = $true,
                HelpMessage = "Enter computer name(s)"
        )]
        [Alias('CN', 'MachineName')]
        [ValidateNotNullOrEmpty()]
        [string[]]$ComputerName,

        [Parameter()]
        [ValidateRange(1, 3600)]
        [int]$TimeoutSeconds = 60,

        [Parameter()]
        [switch]$IncludeDiskInfo
    )

    begin {
        Write-Verbose "Starting health check with timeout: $TimeoutSeconds seconds"
        $results = @()
    }

    process {
        foreach ($computer in $ComputerName)
        {
            if ( $PSCmdlet.ShouldProcess($computer, "Check health"))
            {
                try
                {
                    $cpu = Get-CimInstance -ClassName Win32_Processor -ComputerName $computer -ErrorAction Stop
                    $os = Get-CimInstance -ClassName Win32_OperatingSystem -ComputerName $computer

                    $healthData = [PSCustomObject]@{
                        ComputerName = $computer
                        CPUUsage = $cpu.LoadPercentage
                        MemoryFreeGB = [math]::Round($os.FreePhysicalMemory / 1MB, 2)
                        Uptime = (Get-Date) - $os.LastBootUpTime
                        Timestamp = Get-Date
                    }

                    if ($IncludeDiskInfo)
                    {
                        $disks = Get-CimInstance -ClassName Win32_LogicalDisk -ComputerName $computer -Filter "DriveType=3"
                        $healthData | Add-Member -NotePropertyName DiskInfo -NotePropertyValue $disks
                    }

                    $results += $healthData
                }
                catch
                {
                    Write-Error "Failed to query $computer : $_"
                }
            }
        }
    }

    end {
        Write-Verbose "Completed health check for $( $results.Count ) systems"
        return $results
    }
}

# Usage examples
Get-SystemHealth -ComputerName "Server01" -Verbose
"Server01", "Server02" | Get-SystemHealth -IncludeDiskInfo
Get-SystemHealth -ComputerName "Server01" -WhatIf  # Test without executing
```

## Variable Scope

| Scope       | Description            | Accessibility             |
|-------------|------------------------|---------------------------|
| **Global**  | Session-wide           | Everywhere                |
| **Script**  | Current script file    | Script and its children   |
| **Local**   | Current function/block | Function/block only       |
| **Private** | Current scope only     | Not inherited by children |

### Scope Modifies

```powershell
# Explicit scope declaration
$Global:ConfigPath = "C:\Global\config.json"    # Available everywhere
$Script:Counter = 0                              # Available in this script only
$Local:TempVar = "temporary"                     # Current block only
$Private:Secret = "hidden"                       # Not visible to child scopes

# Best practice: Avoid global variables when possible
function Update-ScriptCounter
{
    $Script:Counter++    # Modify script-level variable
    Write-Host "Counter: $Script:Counter"
}
```

### Scope Hierarchy & Dot-Sourcing

```powershell
# Normal execution (child scope)
.\MyScript.ps1    # Variables in script not available here

# Dot-sourcing (current scope)
. .\MyScript.ps1  # Variables in script ARE available here

# Call operator (child scope)
& ".\MyScript.ps1"  # Explicit child scope execution
```

**Scope Best Practices**

```powershell
# BAD: Polluting global scope
function Set-Config
{
    $Global:Config = @{ Server = "localhost"; Port = 8080 }
}

# GOOD: Return values properly
function Get-Config
{
    return @{ Server = "localhost"; Port = 8080 }
}
$config = Get-Config  # Store in caller's scope

# GOOD: Module-level variables using script: scope
$Script:ModuleCache = @{ }  # Available to all functions in module

function Get-CachedData
{
    param([string]$Key)
    if (-not $Script:ModuleCache.ContainsKey($Key))
    {
        $Script:ModuleCache[$Key] = (Get-ExpensiveData -Key $Key)
    }
    return $Script:ModuleCache[$Key]
}
```

## Complete Dynamic Script Example

```powershell
<#
.SYNOPSIS
    Dynamic system maintenance script with advanced parameters and scope management.
.DESCRIPTION
    Demonstrates variables, operators, conditionals, functions, and scope.
#>
[CmdletBinding(SupportsShouldProcess = $true)]
param(
    [Parameter(Mandatory)]
    [ValidateSet('Analyze', 'Clean', 'Report')]
    [string]$Action,

    [string[]]$ComputerName = $env:COMPUTERNAME,

    [ValidateRange(1, 100)]
    [int]$ThresholdPercent = 80,

    [switch]$AutoFix
)

# Script-level configuration
$Script:LogBuffer = @()
$Script:Results = @()

function Write-Log
{
    param([string]$Message, [ValidateSet('Info', 'Warning', 'Error')][string]$Level = 'Info')
    $entry = "[{0:HH:mm:ss}] [{1}] {2}" -f (Get-Date), $Level, $Message
    $Script:LogBuffer += $entry

    $colorMap = @{ Info = 'White'; Warning = 'Yellow'; Error = 'Red' }
    Write-Host $entry -ForegroundColor $colorMap[$Level]
}

function Test-DiskSpace
{
    param([string]$Computer)

    try
    {
        $disks = Get-CimInstance -ClassName Win32_LogicalDisk -ComputerName $Computer -Filter "DriveType=3"

        foreach ($disk in $disks)
        {
            $freePercent = [math]::Round(($disk.FreeSpace / $disk.Size) * 100, 2)
            $isLow = $freePercent -lt (100 - $ThresholdPercent)

            [PSCustomObject]@{
                Computer = $Computer
                Drive = $disk.DeviceID
                FreePercent = $freePercent
                Status = if ($isLow)
                {
                    'CRITICAL'
                }
                else
                {
                    'OK'
                }
                NeedsAction = $isLow
            }
        }
    }
    catch
    {
        Write-Log "Failed to query $Computer : $_" -Level Error
    }
}

# Main execution
Write-Log "Starting $Action operation on $( $ComputerName -join ', ' )"

switch ($Action)
{
    'Analyze' {
        foreach ($computer in $ComputerName)
        {
            $Script:Results += Test-DiskSpace -Computer $computer
        }
        $Script:Results | Format-Table -AutoSize
    }

    'Clean' {
        if (-not $AutoFix -and -not $PSCmdlet.ShouldProcess("temp files", "Clean"))
        {
            break
        }

        foreach ($computer in $ComputerName)
        {
            # Cleanup logic here
            Write-Log "Cleaning temp files on $computer"
        }
    }

    'Report' {
        $reportPath = "Report_$( Get-Date -Format 'yyyyMMdd_HHmmss' ).html"
        $Script:Results | ConvertTo-Html -Title "System Report" | Out-File $reportPath
        Write-Log "Report saved to $reportPath"
    }
}

Write-Log "Operation completed. Processed $( $Script:Results.Count ) items."
```
