## Arrays

- Multi Dimensional Arrays
- Jagged Arrays (Arrays of Arrays)

  Unlike true multi-dimensional arrays, jagged arrays can have varying inner array sizes

```powershell

# Standard array (fixed size after creation)
$fruits = @('Apple', 'Banana', 'Cherry')
$numbers = 1, 2, 3, 4, 5

# 2D Array (matrix)
$matrix = @(
    @(1, 2, 3),
    @(4, 5, 6),
    @(7, 8, 9)
)

# Access elements
$matrix[0][1]  # Returns 2 (row 0, column 1)

# 3D Array
$cube = @(
    @(@(1, 2), @(3, 4)),
    @(@(5, 6), @(7, 8))
)
$cube[1][0][1]  # Returns 6

#Jagged Array
$jagged = @(
    @(1, 2, 3, 4, 5), # 5 elements
    @(10, 20), # 2 elements
    @(100, 200, 300)        # 3 elements
)

# Iterate through jagged array
foreach ($row in $jagged)
{
    "Row has $( $row.Count ) elements: $( $row -join ', ' )"
}


# Server farm status matrix
$serverFarm = @(
    @{ Name = 'Web01'; Status = 'Running'; CPU = 45; Memory = 60 },
    @{ Name = 'Web02'; Status = 'Running'; CPU = 62; Memory = 75 },
    @{ Name = 'DB01'; Status = 'Stopped'; CPU = 0; Memory = 0 }
)

# Access and manipulate
foreach ($server in $serverFarm)
{
    $statusColor = if ($server.Status -eq 'Running')
    {
        'Green'
    }
    else
    {
        'Red'
    }
    Write-Host "$( $server.Name ): $( $server.Status ) (CPU: $( $server.CPU )%)" -ForegroundColor $statusColor
}
```

## Collections: ArrayList vs. Generic List

- ArrayList is Deprecated, Microsoft officially recommends Generic List<T> over ArrayList for all new development.

| Feature           | ArrayList (Legacy)            | Generic List<T> (Recommended) |
|-------------------|-------------------------------|-------------------------------|
| **Type Safety**   | No (stores objects)           | Yes (typed)                   |
| **Performance**   | Slower (boxing/unboxing)      | Faster                        |
| **.Add() Return** | Returns index (must suppress) | Returns void (clean)          |
| **Memory**        | Higher overhead               | Optimized                     |
| **Modern .NET**   | Not recommended               | Best practice                 |

**Generic List Implementation**

```powershell
# String list
$names = [System.Collections.Generic.List[string]]::new()
$names.Add('Alice')      # Clean - no output
$names.Add('Bob')
$names.AddRange(@('Carol', 'David'))

# Typed numeric list (memory efficient)
$scores = [System.Collections.Generic.List[int]]::new()
1..1000000 | ForEach-Object { $scores.Add($_) }

# Mixed type list (use [object] when needed)
$objects = [System.Collections.Generic.List[object]]::new()
$objects.Add('String')
$objects.Add(123)
$objects.Add(Get-Date)
```

**Performance Comparison**

```powershell
# BAD: Array addition (exponentially slow for large collections)
$arr = @()
1..10000 | ForEach-Object { $arr += $_ }  # 5+ seconds!

# GOOD: Generic List (fast)
$list = [System.Collections.Generic.List[int]]::new()
1..10000 | ForEach-Object { $list.Add($_) }  # <100ms

# BEST: Output enumeration (fastest for simple collection)
$result = foreach ($i in 1..10000)
{
    $i
}  # Fastest!
```

## HashTables (Dictionaries)

```powershell
# Basic HashTables

# Creation
$user = @{
    Name = 'John Doe'
    Age = 30
    Department = 'IT'
    IsActive = $true
}

# Access
$user.Name           # John Doe
$user['Department']  # IT

# Modification
$user.Age = 31
$user['Location'] = 'New York'  # Add new key
$user.Remove('IsActive')        # Remove key

# Ordered HashTables

# Maintains insertion order
$config = [ordered]@{
    Server = 'localhost'
    Port = 443
    Timeout = 30
    Retries = 3
}

# Advanced HashTable Operations
# Nested HashTables
$infrastructure = @{
    WebServers = @{
        Prod = @('WebProd01', 'WebProd02')
        Dev = @('WebDev01')
    }
    Database = @{
        Primary = 'SQLProd01'
        Secondary = 'SQLProd02'
        ConnectionString = 'Server=...;Database=...'
    }
}

# Access nested data
$infrastructure.WebServers.Prod[0]  # WebProd01

# Iterate
foreach ($category in $infrastructure.Keys)
{
    Write-Host "`n$category :"
    foreach ($subKey in $infrastructure[$category].Keys)
    {
        $value = $infrastructure[$category][$subKey]
        "  $subKey = $( $value -join ', ' )"
    }
}
```

## For Each (3 Types)

**ForEach Statement (Fastest for in-memory collections)**

```powershell
$services = Get-Service | Select-Object -First 10

foreach ($service in $services)
{
    Write-Host "$( $service.Name ) is $( $service.Status )"
}

# With index (manual tracking)
$index = 0
foreach ($item in $collection)
{
    "$index : $item"
    $index++
}
```

**ForEach-Object Cmdlet (Pipeline-friendly)**

```powershell
Get-Process | ForEach-Object {
    "$( $_.ProcessName ) using $( $_.WorkingSet64 / 1MB ) MB"
}

# Aliases (same command)
Get-Process | ForEach { $_.Name }
Get-Process | % { $_.Name }  # Shortest alias
```

**ForEach-Object -Parallel**

- only for long-running, independent tasks

```powershell
# GOOD USE CASE: Network operations (I/O bound)
$computers = 'Server01', 'Server02', 'Server03', 'Server04'
$computers | ForEach-Object -Parallel {
    Test-Connection -TargetName $_ -Count 2 -Quiet
} -ThrottleLimit 4

# GOOD USE CASE: API calls
$urls = 'https://api1.com', 'https://api2.com', 'https://api3.com'
$results = $urls | ForEach-Object -Parallel {
    Invoke-RestMethod -Uri $_ -TimeoutSec 30
} -ThrottleLimit 3

# BAD USE CASE: Simple operations (overhead > benefit)
1..100 | ForEach-Object -Parallel { $_ * 2 }  # Slower than normal!

# Using $using: for external variables
$timeout = 30
$servers | ForEach-Object -Parallel {
    Test-Connection -TargetName $_ -Count 1 -TimeoutSeconds $using:timeout
} -ThrottleLimit 10
```

- When to use Parallel :
    - I/O operations (API calls, file access, network tests)
    - Long-running independent tasks (>1 second each)
    - CPU-intensive work on multi-core machines
- When not to use Parallel :
    - Simple calculations or string operations
    - Small collections (<100 items)

## For, While, Do-While, Do-Until Loops

**For Loop (Index-based)**

```powershell
# Standard for loop
for ($i = 0; $i -lt 10; $i++) {
    Write-Host "Iteration $i"
}

# Multiple variables
for ($i = 0, $j = 0; $i -lt 10; $i++, $j += 2) {
    "i=$i, j=$j"
}

# Reverse
for ($i = 10; $i -ge 0; $i--) {
    $i
}

# Practical: Process array with index
$files = Get-ChildItem *.log
for ($i = 0; $i -lt $files.Count; $i++) {
    "[$i] $( $files[$i].Name ) - $( $files[$i].Length ) bytes"
}
```

**While Loop (Pre-condition)**

```powershell
# Basic while
$counter = 0
while ($counter -lt 5)
{
    Write-Host "Count: $counter"
    $counter++
}

# Practical: Retry logic with backoff
$maxRetries = 3
$retryCount = 0
$success = $false

while (-not $success -and $retryCount -lt $maxRetries)
{
    try
    {
        Invoke-RestMethod -Uri 'https://api.example.com/data' -ErrorAction Stop
        $success = $true
        Write-Host "Success on attempt $( $retryCount + 1 )"
    }
    catch
    {
        $retryCount++
        if ($retryCount -lt $maxRetries)
        {
            $delay = [math]::Pow(2, $retryCount)  # Exponential backoff
            Write-Host "Attempt $retryCount failed. Retrying in $delay seconds..."
            Start-Sleep -Seconds $delay
        }
    }
}
```

**Do-While Loop (Post-condition, runs at least once)**

```powershell
# Ensures execution before condition check
$input = $null
do
{
    $input = Read-Host "Enter 'yes' to continue or 'no' to exit"
} while ($input -ne 'yes' -and $input -ne 'no')
```

**Do-Until Loop (Post-condition, negated logic)**

```powershell
# Runs until condition becomes true
$attempts = 0
do
{
    $attempts++
    Write-Host "Attempt $attempts..."
    $result = Test-Connection -TargetName 'server' -Count 1 -Quiet
} until ($result -or $attempts -ge 5)

if ($result)
{
    "Connected successfully in $attempts attempts"
}
else
{
    "Failed to connect after $attempts attempts"
}
```

## Loop Control: Break, Continue, Return, Exit

```powershell
# Break: Exit loop entirely
foreach ($item in $collection)
{
    if ($item -eq 'stop')
    {
        break
    }
    Write-Host $item
}

# Continue: Skip to next iteration
foreach ($number in 1..20)
{
    if ($number % 2 -eq 0)
    {
        continue
    }  # Skip even numbers
    Write-Host $number  # Only odd numbers
}

# Return: Exit function/script with value (in loops within functions)
function Find-FirstMatch
{
    foreach ($item in $args)
    {
        if ($item -like '*target*')
        {
            return $item  # Exits function, returns value
        }
    }
    return $null
}

# Exit: Terminate entire script
foreach ($config in $configs)
{
    if (-not (Test-Path $config.RequiredFile))
    {
        Write-Error "Critical file missing: $( $config.RequiredFile )"
        exit 1  # Terminate with error code
    }
}
```

## Array Slicing and Manipulation

```powershell
$arr = 0..9

# Slicing (PowerShell 7+)
$arr[2..5]        # Elements 2 through 5
$arr[-3..-1]     # Last 3 elements
$arr[0, 2, 4, 6]    # Specific indices

# Reverse
$arr[-1..-($arr.Count)]

# Where method (PowerShell 4+)
$arr.Where({ $_ -gt 5 }, 'Default')   # All > 5
$arr.Where({ $_ -gt 5 }, 'First')     # First > 5
$arr.Where({ $_ -gt 5 }, 'Last')      # Last > 5
$arr.Where({ $_ -gt 5 }, 'SkipUntil') # Skip until > 5
```

## Queue and Stack Collections

```powershell
# Queue (FIFO - First In, First Out)
$queue = [System.Collections.Generic.Queue[string]]::new()
$queue.Enqueue('Task1')
$queue.Enqueue('Task2')
$queue.Enqueue('Task3')
$nextTask = $queue.Dequeue()  # Task1

# Stack (LIFO - Last In, First Out)
$stack = [System.Collections.Generic.Stack[int]]::new()
$stack.Push(10)
$stack.Push(20)
$stack.Push(30)
$last = $stack.Pop()  # 30
```

## Example

```powershell
# Server inventory with multi-dimensional data and efficient collection handling
function Get-ServerInventory
{
    [CmdletBinding()]
    param(
        [string[]]$ComputerName,
        [int]$ThrottleLimit = 5
    )

    # Use Generic List for results (efficient)
    $results = [System.Collections.Generic.List[hashtable]]::new()

    # Process servers in parallel if many
    if ($ComputerName.Count -gt 10)
    {
        $ComputerName | ForEach-Object -Parallel {
            $info = @{
                Computer = $_
                Status = 'Unknown'
                Services = @()
                Disks = @()
                Timestamp = Get-Date
            }

            if (Test-Connection -TargetName $_ -Count 1 -Quiet)
            {
                $info.Status = 'Online'
                $info.Services = @(Get-Service -ComputerName $_ |
                        Where-Object { $_.Status -eq 'Running' } |
                        Select-Object -First 5 Name, DisplayName)
                $info.Disks = @(Get-CimInstance -ComputerName $_ -ClassName Win32_LogicalDisk |
                        Where-Object { $_.DriveType -eq 3 } |
                        Select-Object DeviceID,
                        @{ N = 'FreeGB'; E = { [math]::Round($_.FreeSpace/1GB, 2) } },
                        @{ N = 'SizeGB'; E = { [math]::Round($_.Size/1GB, 2) } })
            }
            else
            {
                $info.Status = 'Offline'
            }

            $info
        } -ThrottleLimit $ThrottleLimit | ForEach-Object {
            $results.Add($_)
        }
    }
    else
    {
        # Sequential processing for small sets (less overhead)
        foreach ($computer in $ComputerName)
        {
            $info = @{
                Computer = $computer
                Status = 'Unknown'
                Services = @()
                Disks = @()
                Timestamp = Get-Date
            }

            if (Test-Connection -TargetName $computer -Count 1 -Quiet)
            {
                $info.Status = 'Online'
                $info.Services = @(Get-Service -ComputerName $computer |
                        Where-Object { $_.Status -eq 'Running' } |
                        Select-Object -First 5 Name, DisplayName)
                $info.Disks = @(Get-CimInstance -ComputerName $computer -ClassName Win32_LogicalDisk |
                        Where-Object { $_.DriveType -eq 3 } |
                        Select-Object DeviceID,
                        @{ N = 'FreeGB'; E = { [math]::Round($_.FreeSpace/1GB, 2) } },
                        @{ N = 'SizeGB'; E = { [math]::Round($_.Size/1GB, 2) } })
            }
            else
            {
                $info.Status = 'Offline'
            }

            $results.Add($info)
        }
    }

    # Convert to array for output (standard PowerShell behavior)
    return $results.ToArray()
}

# Usage
$servers = 1..20 | ForEach-Object { "Server$_" }
$inventory = Get-ServerInventory -ComputerName $servers -ThrottleLimit 8

# Analyze results
$inventory | Group-Object -Property Status |
        Select-Object Name, Count, @{ N = 'Servers'; E = { $_.Group.Computer -join ', ' } }
```

## What to use When

| Scenario                                 | Recommended Approach                              |
|------------------------------------------|---------------------------------------------------|
| **Small fixed collection**               | Standard array `@()`                              |
| **Dynamic collection (adding/removing)** | Generic List `[List[T]]`                          |
| **Key-value lookups**                    | HashTable `@{}` or Dictionary `[Dictionary[K,V]]` |
| **FIFO processing**                      | Queue `[Queue[T]]`                                |
| **LIFO processing**                      | Stack `[Stack[T]]`                                |
| **Simple iteration**                     | `foreach` statement                               |
| **Pipeline processing**                  | `ForEach-Object`                                  |
| **Long-running independent tasks**       | `ForEach-Object -Parallel`                        |
| **Multi-dimensional data**               | Array of arrays or array of HashTables            |
| **Ordered data**                         | `[ordered]@{}` or Generic List                    |
