#### LifeCycle

-> new -> [t.start()] -> Ready or Runnable -> [If Thread Scheduler allocates processor ]->
Running -> [If run() method completes then  Thread enters to dead state] ->Dead

- After starting a Thread , if we are trying to start the same thread again then we will get runtime exception.
- If a Runnable object is not passed to the Thread constructor, the Thread’s run() method executes, which does nothing.
  When a Runnable is passed, Thread delegates execution to the Runnable’s run() method.

**Runnable NOT Passed to Thread**

```java
public class ThreadDemo {

    public static void main(String[] args) {

        // Creating Runnable object
        MultiThreadingDemo1 r = new MultiThreadingDemo1();

        // Creating Thread object WITHOUT passing Runnable
        Thread t = new Thread();

        // Starting thread
        t.start();

        // Main thread execution
        System.out.println("Main Thread");
    }
}

// Runnable implementation
class MultiThreadingDemo1 implements Runnable {

    @Override
    public void run() {
        System.out.println("Run method");
    }
}

```

- Because Runnable object is NOT passed
- Thread.run() executes (does nothing)

**Runnable Passed to Thread**

```java
public class ThreadDemo {

    public static void main(String[] args) {

        // Creating Runnable object
        MultiThreadingDemo1 r = new MultiThreadingDemo1();

        // Passing Runnable object to Thread constructor
        Thread t = new Thread(r);

        // Starting thread
        t.start();

        // Main thread execution
        System.out.println("Main Thread");
    }
}

// Runnable implementation
class MultiThreadingDemo1 implements Runnable {

    @Override
    public void run() {
        System.out.println("Run method");
    }
}
```

============

```java
public class ThreadDemo {

    public static void main(String[] args) {

        MultiThreadingDemo1 r = new MultiThreadingDemo1();

        Thread t1 = new Thread();
        Thread t2 = new Thread(r);

        t1.start();   // Calls Thread class start() → new thread → Thread.run() (empty)

        t2.start();   // Calls Thread class start() → new thread → Runnable run()

        t1.run();     // Normal method call → Thread.run()

        t2.run();     // Normal method call → Runnable run()

        // r.start(); // Compile-time error (Runnable has no start())

        r.run();      // Normal method call → Runnable run()

        System.out.println("Main Thread");
    }
}

class MultiThreadingDemo1 implements Runnable {

    @Override
    public void run() {
        System.out.println("Run method");
    }
}
```

| Method Call | New Thread Created? | Which run() Executes? | Executes In |
|-------------|---------------------|-----------------------|-------------|
| t1.start()  | Yes                 | Thread.run()          | New thread  |
| t1.run()    | No                  | Thread.run()          | Main thread |
| t2.start()  | Yes                 | Runnable.run()        | New thread  |
| t2.run()    | No                  | Runnable.run()        | Main thread |
| r.start()   | Compile error       | —                     | —           |
| r.run()     | No                  | Runnable.run()        | Main thread |
